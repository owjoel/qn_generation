Question: What is the first step in the Git Workflow as described in the DevOps Automation lecture notes?
Answer: First, clone code from the project’s shared repository hosted on a server, e.g., GitHub

Question: In the Git Workflow, what is typically done after running unit testing locally?
Answer: Commit to git locally

Question: According to the lecture notes, what is the purpose of creating a branch for a new user story in Git branching and merging?
Answer: To work on the new user story without affecting the main project

Question: What is the next step after fixing an urgent production issue in Git branching and merging, based on the notes?
Answer: Create a new branch "hotfix" based on "master" 

Question: In Continuous Integration implementation, what should be done when the build is broken?
Answer: Fix the build as soon as it’s broken.
